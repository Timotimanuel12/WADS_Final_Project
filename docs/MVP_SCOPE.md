# MVP Scope

This document finalizes the MVP scope for HelpImTooLazy.

## Core MVP Features

1. Smart Scheduling
- Input tasks with due dates, priority, and estimated duration.
- Generate a recommended study plan that avoids obvious overlaps.
- Show prioritized tasks in dashboard/calendar views.

2. Focus Timer
- Pomodoro-style timer (start, pause, reset, complete).
- Record completed focus sessions with duration and timestamp.
- Connect session data to progress analytics.

3. Progress Dashboard
- Show task completion metrics and focus-time totals.
- Provide weekly/monthly trend summaries.
- Highlight urgent tasks and study consistency (streak-oriented view).

## Out of Scope for MVP

- Complex team/collaboration features.
- Deep calendar provider sync (Google/Outlook full sync).
- Advanced AI autonomy beyond assisted prioritization/scheduling.

## MVP Acceptance Summary

- A user can authenticate, manage tasks, run timer sessions, and see personal progress metrics.
- Critical security controls are active on protected data paths.
- Core flows are testable and documented.
